## how to use

it works on Ruby 2.4.0

'$ sudo apt-get install rbenv'
'$ rbenv install 2.4.0'
'$ rbenv global 2.4.0'
'$ sudo gem install bundler'
'$ bundle install'
'$ ruby deepq_client.rb'
